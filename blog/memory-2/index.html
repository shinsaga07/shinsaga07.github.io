記憶碎片 夥伴

記憶碎片 - 夥伴

By 心尋・しんさが 2026-01-19

再前往下一個神殿的途中，我在距離下一個村莊不遠處的森林看到遍地的魔物屍體

更往前一點我看到一人摀著腹部，臉色慘白的依靠著樹

而在他的身邊有蹲著一個人，他正在施展治療魔法試圖止住對方腹部不斷滲血的狀況

但治療的效果看起來不是很順利，施展治療魔法的人顯得相當慌張與不安

「過於嚴重的傷口，一般治療魔法的效果是有限的」

我走到他們身旁這麼說著，並從包袱中拿出一包粉末遞給他們

「這是可以暫時止血的藥粉，先用這個止住血，再到附近的院所進行治療吧」

藥粉撒在傷口上後，血迅速的就止住，原先慌張的人明顯鬆了一口氣

接著便背起受傷的夥伴，準備快點回村莊進行更細部的治療

而我則陪著他們，暫時護衛著他們回到村莊，而在回去的過程中，受傷的人被他的夥伴責備

「為什麼你要一個人來討伐這些魔物」

「你並不需要一個人承擔保護村莊的責任」

「我們不是夥伴嗎，多信任我吧，即便我戰鬥能力不如你，至少我還是能夠在一旁輔助你的」

「其他人受過訓練的人也可以協助你呀！！」

「為什麼你總是要這樣」

「為什麼你不肯信任身旁的人，總是要一肩扛起一切」

類似的訓斥我也曾經聽過

那是在很久很久以前，我與勇者小隊一同旅行時的故事

在夥伴們經歷過一場長久的守衛戰，都累翻熟睡的時候，我感知道早些被擊退的魔族軍隊正帶領著更多、更強的軍團朝我們逼近

看著遍體鱗傷，累癱在地的夥伴們，我沒有叫醒他們，而是獨自一人朝敵人迎去，打算在遠處攔截並擊退他們

然而這次魔族軍團的人數遠超乎我的想像，甚至還有兩名將軍級的統領

就在我即將因為應付不過來眾多的攻擊時，他們幫我擋住了攻擊，並站在我的面前

他們當時的表情相當的難看，我感覺的出他們相當的生氣

◆◆◆：「這已經不是第一次了」

◆◆◆：「我們在你眼中就這麼弱，只能接受你的保護嗎」

因為我失去的太多了，我只想要盡可能的守護我能守護的人們

◆◆◆：「你不必一肩扛起所有責任，過去的事情已經過去，相信現在的夥伴吧」

◆◆◆：「你為什麼不多依賴我們一些呢？」

當時的我沒有辦法做出回應，因為我不知道該說什麼

「多依賴夥伴一些吧，多相信夥伴一些吧」

看著眼前的這兩位年輕人，我不經脫口說出過去◆◆◆和我說過的話

「不要一個人去承擔著一切，這太辛苦了」
